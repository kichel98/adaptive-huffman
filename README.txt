Program jest napisany w języku Java (pisane na wersji 11, wymagana na pewno przynajmniej Java 8).

Implementacja dynamicznej wersji kodowania Huffmana (wersja na ocenę 4).

Kompilacja:
javac com/kikd/lista2/*.java

Uruchomienie:
java com.kikd.lista2.Main --encode file.in file.enc
lub
java com.kikd.lista2.Main --decode file.enc file.dec
